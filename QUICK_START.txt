## ğŸ’¾ DATA STORAGE

### Cart
- Stored in: `localStorage['skh_traders_cart']`

### CSRF Token
- Stored in: `sessionStorage['skh_traders_csrf']`
- Expires: End of browser session
- Format: 64-character hex string

- Format: JSON array of orders
- Includes: ID, timestamp, customer, items, total
